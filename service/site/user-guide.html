Scheme API Portal

API Registration and Discovery made easy.

Uploading API specifications
curl -X POST -H "Content-Type: application/openapi+json" --data-binary @path/to/api/spec http://localhost:8000/api/apis/:id

Supported content types:

application/openapi+json
Supports Swagger and OpenAPI schemas version 2 and 3
application/openapi+yaml
Supports the same versions as JSON conent
application/wadl+xml
Basic WADL support
Searching
curl http://localhost:8000/api/search?query=
Registration

Registration

As an alternative to uploading a specification you can post a registration payload to /registrations referencing an online source for an API specification.

{
    "id": "id for the API",
    "url": "URL for the API specification source",
    "request": {
        "method": "HTTP Method GET/POST",
        "headers": "map of header keys and values",
        "body": "body text to be sent to the server"
    }
}
